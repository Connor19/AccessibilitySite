Button Accessibility

Buttons should be accesssible because otherwise it can cause a disabled user to struggle to read the text or click on.

Bad Example

This example is bad because the button is way to small and it has very small text. This is a problem because it would mean that for disabled users it could be hard to press the button and to read the text.

Good Example

This example is good because the button are not too small or too big meaning the user could click it easier and the text is large enough to read.
