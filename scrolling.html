Scrolling Accessibility

Scrolling should be accessible because if a webpage has a lot of scrolling or a difficulty of scrolling options then disabled users may struggle to scroll down.

Bad Example

This example is bad because the text on the page has been layed out in a way which means the user has to scroll right to keep reading. This massively decreases readadibility and usability.

This example is good because the user wouldnt have to scroll horizontally meaning all content is kept in sight and doesnt require scrolling..
