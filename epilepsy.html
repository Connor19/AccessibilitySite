Epilepsy Accessibility

Epilepsy has to be considered in web design because it is a disabilty and can cause serious reactions in some users.

What you Shouldn't do!

This is what shouldn't happen, when a website has content including flashing lights, it needs to have a warning and an option to pause the content to prevent an epileptic fit.

What you Should do!

This is what should happen, before any content with flashing lights, it should show a warning like this as well an an option to pause the content at any time.
